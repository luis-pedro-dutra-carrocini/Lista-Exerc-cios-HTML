12 - HTML Images (EXTRA)

A tag HTML <img> é usada para incorporar uma imagem em uma página da web. As imagens não são tecnicamente inseridas em uma página da web; as imagens estão vinculadas a páginas da web.
Ela cria um espaço de retenção para a imagem referenciada, contendo apenas atributos e não possui tag de fechamento. Sendo dois deles obrigatórios:

src - Especifica o caminho para a imagem
alt - Especifica um texto alternativo para a imagem


12.1 - Atributo Src e Alt

O atributo obrigatório src especifica o caminho (URL) para a imagem. Quando uma página web carrega, é o navegador, naquele momento, que obtém a imagem de um servidor web e a insere na página. Portanto, certifique-se de que a imagem realmente permaneça no mesmo lugar em relação à página web, caso contrário seus visitantes receberão um ícone de link quebrado. O ícone do link quebrado e o alttexto serão mostrados se o navegador não conseguir encontrar a imagem.

O atributo obrigatório alt fornece um texto alternativo para uma imagem, se o usuário por algum motivo não puder visualizá-la (por causa de uma conexão lenta, um erro no atributo src ou se o usuário usar um leitor de tela). O valor do alt deve descrever a imagem, pois se um navegador não conseguir encontrar uma imagem, ele exibirá o seu valor:

Código:

<img src="flores_china.jpg" alt="Flores na China">

12.2 - Tamanho da imagem - Width e Height

Os atributos widthe height sempre definem a largura e a altura da imagem em pixels.
Você pode usar o style atributo para especificar a largura e a altura de uma imagem. Ou você pode usar os atributos widthe e height:

Código:

<img src="img/walle.jpg" alt="Imagem do Walle do desenho" style="width:500px; height:600px;"> <img src="img/walle.jpg" alt="Imagem do Walle do desenho" width="500" height="600">

Exemplo:


Os atributos width, heighte style são todos válidos em HTML. No entanto, se usar o atributo style, as folhas de estilos não alteraram o tamanho das imagens.

12.3 - Imagens em outra pasta

Se você tiver suas imagens em uma subpasta, deverá incluir o nome da pasta no atributo src:

Código:

<img src="img/css.gif" alt="CSS Gif" style="width:150px; height:150px;">

12.4 - Imagens em outro servidor/site

Alguns sites apontam para uma imagem em outro servidor. Para apontar para uma imagem em outro servidor, você deve especificar uma URL absoluta (completa) no atributo src:

12.6 - Mapas de imagens

A tag HTML <map> define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis. As áreas são definidas com uma ou mais <area> tags.
Tente clicar no computador, telefone ou na xícara de café da imagem abaixo:

Código:

<img src="img/workplace.jpg" alt="Workplace" usemap="#workmap">
<map name="workmap">
<area shape="rect" coords="34,44,270,350" alt="Computador" href="exemplos/computador.html">
<area shape="rect" coords="290,172,333,250" alt="Telefone" href="https://www.w3schools.com/html/html_images_imagemap.asp">
<area shape="circle" coords="337,300,44" alt="Chícara" href="exemplos/chicara.html">
</map>

Exemplo:

Use o elemento <map> HTML para definir um mapa de imagem.
Use o elemento <area> HTML para definir as áreas clicáveis no mapa de imagem.
Use o atributo usemap HTML do <img> elemento para apontar para um mapa de imagem.
Você deve definir o formato da área clicável (shap) e pode escolher um destes valores:

rect- define uma região retangular
circle- define uma região circular
poly- define uma região poligonal
default- define toda a região


12.7 - Imagem de fundo em um elemento HTML

Se a imagem de fundo for menor que o elemento, a imagem se repetirá, horizontal e verticalmente, até chegar ao final do elemento:

Código:

<div class="explicacao-tag" style="background-image: url('img/the_rock.jpg'); height: 300px;"></div>

Exemplo:


Para evitar que a imagem de fundo se repita, defina a propriedade background-repeat como no-repeat.
Se quiser que a imagem de fundo cubra todo o elemento, você pode definir a propriedade background-size como cover.

Código:

<div style="background-image: url('img/the_rock.jpg'); background-repeat: no-repeat; background-attachment: fixed; background-size: cover; width: 300px; height: 300px;"></div>
