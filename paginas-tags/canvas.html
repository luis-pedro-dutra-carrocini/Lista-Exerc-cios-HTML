35 - HTML Canvas

35.1 - Explicação Canvas

O elemento HTML <canvas> é usado para desenhar gráficos em uma página da web. É usado para desenhar gráficos dinamicamente, via JavaScript. Ele é apenas um contêiner para gráficos. Você deve usar JavaScript para realmente desenhar os gráficos. O Canvas possui vários métodos para desenhar caminhos, caixas, círculos, texto e adicionar imagens. Canvas é compatível com todos os principais navegadores.


35.2 - Exemplos de tela

Uma tela é uma área retangular em uma página HTML. Por padrão, uma tela não tem borda nem conteúdo.

Sempre especifique um idatributo (a ser referenciado em um script) e um atributo width e height para definir o tamanho da tela. Para adicionar uma borda, use o atributo style.

Código:

<canvas id="myCanvas" width="200" height="100" style="border: 2px solid black;"></canvas>

35.3 - Adicione um JavaScript

Depois de criar a área retangular da tela, você deve adicionar um JavaScript para fazer o desenho.

Aqui estão alguns exemplos:

Desenhe uma linha:

<canvas id="canvas_reta" width="200" height="100"></canvas>
<script>
   var c = document.getElementById("canvas_reta");
   var ctx = c.getContext("2d");
   ctx.moveTo(0, 0);
   ctx.lineTo(200, 100);
   ctx.stroke();
</script>

Desenhar um círculo:

<canvas id="canvas_circulo" width="200" height="100"></canvas>
<script>
   var c = document.getElementById("canvas_circulo");
   var ctx = c.getContext("2d");
   ctx.beginPath();
   ctx.arc(95, 50, 40, 0, 2 * Math.PI);
   ctx.stroke();
</script>

Resultado:


Desenhe um texto:

<canvas id="canvas_frase" width="200" height="100"></canvas>
<script>
   var c = document.getElementById("canvas_frase");
   var ctx = c.getContext("2d");
   ctx.font = "30px Arial";
   ctx.fillText("Hello World", 10, 50);
</script>

Resultado:


Texto de traço:

<canvas id="canvas_texto_traco" width="200" height="100"></canvas>
<script>
   var c = document.getElementById("canvas_texto_traco");
   var ctx = c.getContext("2d");
   ctx.font = "30px Arial";
   ctx.strokeText("Hello World", 10, 50);
</script>

Resultado:


Desenhar Degrade linear:

<canvas id="canvas_degrade" width="200" height="100"></canvas>
<script>
   var c = document.getElementById("canvas_degrade");
   var ctx = c.getContext("2d");

   // Criando Degrade
   var grd = ctx.createLinearGradient(0, 0, 200, 0);
   grd.addColorStop(0, "blue");
   grd.addColorStop(1, "white");

   // Preenchendo com o Degrade
   ctx.fillStyle = grd;
   ctx.fillRect(10, 10, 150, 80);
</script>

Resultado:


Desenhar Degrade circular:

<canvas id="canvas_degrade_circular" width="200" height="100"></canvas>
<script>
   var c = document.getElementById("canvas_degrade_circular");
   var ctx = c.getContext("2d");

   // Criando Degrade

   var grd = ctx.createRadialGradient(75, 50, 5, 90, 60, 100);
   grd.addColorStop(0, "green");
   grd.addColorStop(1, "white");

   // Preenchendo com o Degrade
   ctx.fillStyle = grd;
   ctx.fillRect(10, 10, 150, 80);
</script>
