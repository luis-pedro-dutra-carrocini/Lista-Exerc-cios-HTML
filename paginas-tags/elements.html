2 - HTML Elements

Estrutura:

A grande maioria das tags precisão ser abertas e posteriormente fechadas: <nometag> Conteudo da TAG </nometag>

Início da TAG Conteudo da TAG Fim da TAG
h1 <h1> </h1>
p <p> </p>
<br> Não Possui Não Possui

2.1 - Títulos HTML

Os títulos HTML são definidos com as tags <h1> a <h6>.
<h1> define o título maior. <h6> define o título menor.

Código:

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>

Exemplo:

Título 1

Título 2

Título 3


2.2 - Parágrafos HTML

Os parágrafos HTML são definidos com a tag <p>.

Código:

<p>Conteudo do Paragrafo</p>

Exemplo:

Conteudo do Paragrafo


2.3 - Parágrafos com quebra de linha HTML

Para se quebrar uma linha basta adicionar a tag <br> onde deseja que a linha seja quebrada.

Código:

<p>Conteudo do Paragrafo <br> Outro Conteudo </p>

Exemplo:

Conteudo do Paragrafo
Outro Conteudo


2.4 - Links HTML

Os links HTML são definidos com a tag <a>.

Código:

<a href="https://www.exemplo.com">Link a ser clicado</a>

Exemplo:

Link a ser clicado

2.5 - Imagens HTML

As Imagens HTML são exibidas com a tag <img>.

Código:

<img src="imagem.png" alt="Descrição da Imagem" width="350" height="250">
