28 - HTML vs XHTML

28.1 - O que é XHTML?

XHTML é uma versão do HTML mais rígida e baseada em XML. Significa E X tensible Hyper Text Markup Language , é HTML definido como um aplicativo XML e ele é suportado por todos os principais navegadores.

XML é uma linguagem de marcação onde todos os documentos devem ser marcados corretamente (ser "bem formados"). Ele foi desenvolvido para tornar o HTML mais extensível e flexível para trabalhar com outros formatos de dados (como XML). Além disso, os navegadores ignoram erros nas páginas HTML e tentam exibir

28.2 - As diferenças mais importantes do HTML

  • <!DOCTYPE> é obrigatório
  • O atributo xmlns em <html> é obrigatório
  • <html>, <head>, <title> e <body> são obrigatórios
  • Os elementos devem sempre estar adequadamente aninhados
  • Os elementos devem estar sempre fechados
  • Os elementos devem estar sempre em letras minúsculas
  • Os nomes dos atributos devem estar sempre em letras minúsculas
  • Os valores dos atributos devem sempre ser citados
  • A minimização de atributos é proibida

28.3 - Aqui está um documento XHTML com um mínimo de tags obrigatórias:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN"
"http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title>Título do documento</title>
</head>
<body>

conteudo da página....

</body>
</html>

28.4 - Elementos XHTML devem ser aninhados corretamente

Correto:

<b><i>Conteúdo</i></b>

Errado:

<b><i>Conteúdo</b></i>

28.5 - Elementos XHTML devem sempre ser fechados

Correto:

<p>Isso é um parágrafo</p>
<p>Contunuação do parágrafo</p>

Errado:

<p>Isso é um parágrafo
<p>Contunuação do parágrafo

28.6 - Elementos vazios XHTML devem sempre ser fechados

Correto:

Um br: <br />
Um hr: <hr />
Uma imagem: <img src="happy.gif" alt="Happy face" />

Errado:

Um br: <br>
Um hr: <hr>
Uma imagem: <img src="happy.gif" alt="Happy face">

28.7 - Os elementos XHTML devem estar em letras minúsculas

Correto:

<body>
<p>Isso é um parágrafo</p>
</body>

Errado:

<BODY>
<P>Isso é um parágrafo</P>
</BODY>

28.8 - Os nomes dos atributos XHTML devem estar em letras minúsculas

Correto:

<a href="https://www.w3schools.com/html/">Visite o nosso tutorial de HTML</a>

Errado:

<a HREF="https://www.w3schools.com/html/">Visite o nosso tutorial de HTML</a>

28.9 - Valores de atributos XHTML devem ser citados entre aspas

Correto:

<a href="https://www.w3schools.com/html/">Visite o nosso tutorial de HTML</a>

Errado:

<a href=https://www.w3schools.com/html/>Visite o nosso tutorial de HTML</a>

28.9 - A minimização de atributos XHTML é proibida

Correto:

<input type="checkbox" name="vehicle" value="car" checked="checked" />
<input type="text" name="lastname" disabled="disabled" />

Errado:

<input type="checkbox" name="vehicle" value="car" checked />
<input type="text" name="lastname" disabled />
