4 - HTML Headings

Títulos HTML

Os títulos HTML são definidos com as tags <h1> a <h6>.

<h1> define o título maior. <h6> define o título menor.

Código:

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h1>Título 1</h4>
<h1>Título 1</h5>
<h1>Título 1</h6>
