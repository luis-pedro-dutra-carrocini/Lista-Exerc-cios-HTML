36.1 - SVG (Gráficos Vetoriais Escaláveis)

SVG define gráficos baseados em vetores em XML, que podem ser incorporados diretamente em páginas HTML. Os gráficos SVG são escaláveis ​e não perdem qualidade se forem ampliados ou redimensionados. Ele é compatível com todos os principais navegadores.


O que é?

  • SVG significa gráficos vetoriais escaláveis
  • SVG é usado para definir gráficos baseados em vetores para a Web
  • SVG define gráficos em formato XML
  • Cada elemento e atributo em arquivos SVG pode ser animado
  • SVG é uma recomendação do W3C
  • SVG integra-se com outros padrões, como CSS, DOM, XSL e JavaScript

36.2 - O elemento <svg>

O elemento HTML <;svg> é um contêiner para gráficos SVG. Ele possui vários métodos para desenhar caminhos, retângulos, círculos, polígonos, texto e muito mais.

Cículo:

<svg width="100" height="100">
   <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
</svg>

36.3 - Diferenças entre SVG e Canvas

SVG é uma linguagem para descrever gráficos 2D em XML, enquanto o Canvas desenha gráficos 2D dinamicamente (com JavaScript).

SVG é baseado em XML, o que significa que todos os elementos estão disponíveis no SVG DOM. Você pode anexar manipuladores de eventos JavaScript a gráficos SVG.

No SVG, cada forma desenhada é lembrada como um objeto. Se os atributos de um objeto SVG forem alterados, o navegador poderá renderizar novamente a forma automaticamente.

A tela é renderizada pixel por pixel. No canvas, uma vez desenhado o gráfico, ele é esquecido pelo navegador. Se a sua posição for alterada, toda a cena precisa ser redesenhada, incluindo quaisquer objetos que possam ter sido cobertos pelo gráfico.


36.4 - Comparação de SVG e Canvas

SVG Canvas
  • Resolução independente
  • Suporte para manipuladores de eventos
  • Bons recursos de renderização de texto
  • Renderização lenta se complexa
  • Não adequado para aplicações de jogos
  • Dependente da resolução
  • Sem suporte para manipuladores de eventos
  • Capacidades de renderização de texto ruins
  • Você pode salvar a imagem resultante como .png ou .jpg
  • Adequado para jogos com muitos gráficos
