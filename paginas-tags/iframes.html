19 - HTML Iframe

19.1 - Sintaxe

A tag HTML <iframe> especifica um quadro embutido.

Um quadro embutido é usado para incorporar outro documento no documento HTML atual.

É uma boa prática sempre incluir um titleatributo para o arquivo <iframe>. Isso é usado por leitores de tela para ler qual é o conteúdo do iframe.

Um iframe pode ser usado como quadro de destino para um link. O targetatributo do link deve referir-se ao atributo name do iframe.

Sintaxe: <iframe src="url" title="description"></iframe>

Código:

<iframe src="exemplos/capitulos.html" style="height:200px;width:500px;" title="Iframe Exemplo"></iframe><br>
