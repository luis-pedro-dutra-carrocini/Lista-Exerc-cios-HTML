38 - HTML Video

38.1 - O elemento HTML <vídeo>

O elemento HTML <;video> é usado para mostrar um vídeo em uma página da web.

O controlsatributo adiciona controles de vídeo, como reproduzir, pausar e volume.

É uma boa ideia sempre incluir os atributos width e height. Se a altura e a largura não estiverem definidas, a página poderá piscar enquanto o vídeo é carregado.

O elemento <source> permite que você especifique arquivos de vídeo alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

O texto entre as tags <video> e </video> só será exibido em navegadores que não suportam o elemento <video>.

Código:

<video width="320" height="240" controls>
   <source src="img/Ambulância_Besta_Chaves.mp4" type="video/mp4">
   Seu navegador não suporta vídeos.
</video>

38.2 - Atributo autoplay

Para iniciar um vídeo automaticamente, use o atributo autoplay. Os navegadores Chromium não permitem a reprodução automática na maioria dos casos. No entanto, a reprodução automática silenciada é sempre permitida.Adicione muted depois autoplay para permitir que seu vídeo comece a ser reproduzido automaticamente (mas sem som):

Código:

<video width="320" height="240" controls autoplay muted>
   <source src="img/SeuMadruga_MenosAnimal.mp4" type="video/mp4">
   Seu navegador não suporta vídeos.
</video>

37.3 - Formatos multimídia

Elementos multimídia (como áudio ou vídeo) são armazenados em arquivos de mídia. A maneira mais comum de descobrir o tipo de um arquivo é observar a extensão do arquivo. Os arquivos multimídia possuem formatos e extensões diferentes como: .wav, .mp3, .mp4, .mpg, .wmv e .avi.


37.4 - Formatos de vídeo comuns

Existem muitos formatos de vídeo por aí. Os formatos MP4, WebM e Ogg são suportados por HTML. O formato MP4 é recomendado pelo YouTube.
