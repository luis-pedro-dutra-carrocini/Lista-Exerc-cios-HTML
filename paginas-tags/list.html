14 - HTML List

Listas HTML

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

Lista não ordenada

Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li>.

Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

Código Básico:

<p>Opções de desjejum:</p>
<ul>
<li>Café</li>
<li>Chá</li>
<li>Leite</li>
</ul>

Exemplo:

Opções de desjejum:

  • Café
  • Chá
  • Leite

Lista ordenada

Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.

Os itens da lista serão marcados com números por padrão:

Código Básico:

<p>Opções de desjejum:</p>
<ol>
<li>Café</li>
<li>Chá</li>
<li>Leite</li>
</ol>

Exemplo:

Opções de desjejum:

  1. Café
  2. Chá
  3. Leite

Lista de descrição

Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.

Os itens da lista serão marcados com números por padrão:

Código Básico:

<p>Opções de desjejum:</p>
<dl>
<dt>Café</dt>
<dd>- Bebida preta e quente</dd>
<dt>Leite</dt>
<dd>- Bebida branca e fria</dd>
</dl>

Exemplo:

Opções de desjejum:

Café
- Bebida preta e quente
Leite
- Bebida branca e fria
