3 - HTML Attributes

3.1 - Atributo HREF

Esse atributo define a URL da página a qual ele redirecionará, geralmente é utilizado com a tag <a>.

Código:

<a href="https://www.exemplo.com">Link a ser clicado</a>

Exemplo:

Link a ser clicado

3.2 - Atributo SRC / Width e Height

O atributo SRC define o caminho de um arquivo, imagem, etc. Geralmente é utilizado com a tag <img> para definir o caminho da imagem.

Os atributos Width e Height definem a largura e a altura, respectivamente.

Já o atributo ALT exibe um texto alternativo, caso a imagem não seje exibida por algum motivo.

Código:

<img src="imagem.png" alt="Descrição da Imagem" width="350" height="250">

3.5 - Atributo Lang

Você deve sempre incluir o langatributo dentro da tag <html>, para declarar o idioma da página web. O objetivo é ajudar os mecanismos de pesquisa e navegadores.

Os códigos de país também podem ser adicionados ao código do idioma no lang atributo. Assim, os dois primeiros caracteres definem o idioma da página HTML e os dois últimos caracteres definem o país.

O exemplo a seguir especifica o inglês como idioma e Estados Unidos como país.

Código:

<html lang="en-US">

3.6 - Atributo Title

O atributo title define algumas informações extras sobre um elemento. O valor do atributo a tag title será exibido como uma dica de ferramenta quando você passar o mouse sobre o elemento.

Código:

<p title="Eu sou um paragrafo">Texto a ser digitado.</p>

Exemplo:

Texto a ser digitado.


3.7 - Recomendações Gerais

Por padrão o HTML não tem restrições de como os atributos devem ser escritos (Letras Maiúscula ou Letras Minúsculas), no entanto recomendamos que os atributos sejam escritos em letras minúsculas para tipos de documentos mais rígidos, como XHTML.

Também sugerimos sempre colocar os valores dos atributos entre aspas, podem ser aspas simples ou duplas, não importa

Bom:

<a href="https://www.w3schools.com">Entre aspas.</p>

Ruim:

<a href=https://www.w3schools.com>Sem aspas.</a>
