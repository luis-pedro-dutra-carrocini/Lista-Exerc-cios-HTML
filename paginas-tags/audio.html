39 - HTML Audio

39.1 - O elemento HTML <audio>

Para reproduzir um arquivo de áudio em HTML, use o elemento <audio>.

O atributo controls adiciona controles de áudio, como reproduzir, pausar e volume.

O elemento <source> permite que você especifique arquivos de áudio alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

O texto entre as tags <audio> e </audio> só será exibido em navegadores que não suportam o elemento

Código:

<audio controls>
   <source src="img/SeuMadruga_Nossa.mp3" type="audio/mpeg">
   Seu navegador não suporta áudios.
</audio>

39.2 - Atributo autoplay

Para iniciar um arquivo de áudio automaticamente, use o atributo autoplay. Os navegadores Chromium não permitem a reprodução automática na maioria dos casos. No entanto, a reprodução automática silenciada é sempre permitida. Adicione muted depois autoplay para permitir que seu vídeo comece a ser reproduzido automaticamente (mas sem som):

Código:

<audio width="320" height="240" controls autoplay muted>
   <source src="img/nao.mp3" type="audio/mp4">
   Seu navegador não suporta áudios.
</audio>

39.3 - Tipos de mídia

Formato do Arquivo Media Type
MP3 audio/mpeg
OGG audio/ogg
WAV audio/wav

39.4 - Métodos, Propriedades e Eventos

O HTML DOM define métodos, propriedades e eventos para o elemento <audio>.

Isso permite carregar, reproduzir e pausar áudios, bem como definir duração e volume. Existem também eventos DOM que podem notificá-lo quando um vídeo começa a ser reproduzido, é pausado, etc.
