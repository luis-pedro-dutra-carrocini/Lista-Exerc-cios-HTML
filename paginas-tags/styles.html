6 - HTML Styles

Estilos HTML

O atributo HTML style é usado para adicionar características/estilos a um elemento, como cor, fonte, tamanho e muito mais. Segue a estrutura:

<nometag style='propriedade: valor;'>

6.1 - Cor de Fundo

Código:

<h1 style='background-color: blue;'>Texto com o fundo em Azul</h1>

<p style='background-color: red;'>Texto com o fundo em Vermelho</p>

Exemplo:

Texto com o fundo em Azul

Texto com o fundo em Vermelho


6.2 - Cor do Texto

Código:

<h1 style='color: blue;'>Texto em Azul</h1>

<p style='color: red;'>Texto em Vermelho</p>

Exemplo:

Texto em Azul

Texto em Vermelho


6.3 - Fonte

Fonte:

<h1 style='color: blue;'>Texto em Azul</h1>

<p style='color: red;'>Texto em Vermelho</p>

Exemplo:

Texto na fonte Verdana

Texto na fonte Courier


6.4 - Tamanho do Texto

Código:

<h1 style='font-size: 300%;'>Tamanho em 300%</h1>

<p style='font-size: 160%;'>Tamanho em 160%</p>

Exemplo:

Tamanho em 300%

Tamanho em 160%


6.5 - Alinhamento do Texto

Código:

<h1 style="text-align: center;">Texto alinhado no centro.</h1>

<p style="text-align: right;">Texto alinhado para a direita.</p>

Exemplo:

Texto alinhado no centro.

Texto alinhado para a direita.


CSS

Cascading Style Sheets (CSS) é usado para formatar o layout de uma página da web.
Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre os elementos, como os elementos são posicionados e dispostos, quais imagens ou cores de fundo devem ser usadas, diferentes exibições para diferentes dispositivos e tamanhos de tela, e muito mais!
CSS pode ser adicionado a documentos HTML de 3 maneiras:

Inline - usando o atributo style dentro dos elementos HTML
Interno - usando um elemento <style> na seção <head>
Externo - usando um elemento <link> para vincular a um arquivo CSS externo

Vamos ver cada uma delas:

6.6 - Inline CSS

O CSS embutido é usado para aplicar um estilo único a um único elemento HTML. Usa-se o atributo style de um elemento HTML.
O exemplo a seguir define a cor do texto do elemento <h1> como azul e a cor do texto do elemento <p> como vermelho:

Código:

<h1 style="color:blue;">Cabeçalho em Azul</h1>

<p style="color:red;">Parágrafo em Vermelho</p>

Exemplo:

Cabeçalho em Azul

Parágrafo em Vermelho


6.7 - Internal CSS

O CSS interno é usado para definir um estilo para uma única página HTML, é definido na seção de uma página HTML, dentro de um elemento <style>. O exemplo a seguir define a cor do texto de TODOS os elementos <h1> (nessa página) como azul e a cor do texto de TODOS os elementos <p> como vermelho:

Código:

<!DOCTYPE html>
<html>
<head>
<style>
h1 {color: blue;}
p {color: red;}
</style>
</head>
<body>
<h1>Isso é um Cabeçalho</h1>
<p>Isso é um Parágrafo</p>
</body>
</html>

6.8 - External CSS

Uma folha de estilo externa é usada para definir o estilo de muitas páginas HTML.
Para usar uma folha de estilo externa, adicione um link para ela na seção <head> de cada página HTML:

Código no arquivo HTML:

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<h1>Isso é um Cabeçalho</h1>
<p>Isso é um Parágrafo<p>
</body>
</html>

O arquivo não deve conter nenhum código HTML e deve ser salvo com extensão .css.
Esta é a aparência do arquivo “styles.css”:

Código no arquivo CSS:

body {
background-color: powderblue;
}
h1 {
color: blue;
}
p {
color: red;
}
