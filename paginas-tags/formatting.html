7 - HTML Formatting

Formatação de texto HTML

O HTML contém vários elementos que servem para formatar o texto. Trabalharemos com alguns deles:

<b> - Texto em Negrito
<strong> - Texto Importante
<i> - Texto em Itálico
<em> - Texto Enfatizado
<mark> - Texto Marcado
<small> - Texto Menor
<del> - Texto Excluído
<ins> - Texto Inserido
<sub> - Texto Subscrito
<sup> - Texto Sobrescrito

7.1 - Elementos <b> e <strong>

Código:

<b>Texto em Negrito</b>
<strong>Texto Importante!</strong>

Exemplo:

Texto em Negrito
Texto Importante!

7.2 - Elementos <i> e <em>

Código:

<i>Texto em Itálico</i>
<em>Texto Enfatizado!</em>

Exemplo:

Texto em Itálico
Texto Enfatizado!

7.3 - Elemento <small>

Código:

<small>Texto Pequeno</small>

Exemplo:

Texto Pequeno

7.4 - Elemento <mark>

Código:

<p>Texto normal <mark>agora texto marcado</mark> de volta ao texto normal.</p>

Exemplo:

Texto normal, agora texto marcado, de volta ao texto normal.


7.5 - Elemento <del>

Código:

<p>Minha cor favorita é <del>blue</del> red.</p>

Exemplo:

Minha cor favorita é blue red.


7.6 - Elemento <ins>

Código:

<p>Minha cor favorita é <del>blue</del> <ins>red</ins>.</p>

Exemplo:

Minha cor favorita é blue red.


7.8 - Elemento <sub>

Código:

<p>Esse texto está <sub>subscrito</sub> agora não.</p>

Exemplo:

Esse texto está subscrito agora não.


7.9 - Elemento <sup>

Código:

<p>Esse texto está <sup>superescrito</sup> agora não.</p>

Exemplo:

Esse texto está superescrito agora não.
