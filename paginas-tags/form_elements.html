31 - HTML Form Elements

31.1 - Os elementos HTML <form>

O elemento HTML <form> pode conter um ou mais dos seguintes elementos de formulário:

  • <input>
  • <label>
  • <select>
  • <textarea>
  • <button>
  • <fieldset>
  • <legend>
  • <datalist>
  • <output>
  • <option>
  • <optgroup>

31.2 - O elemento <input>

Um dos elementos de formulário mais utilizados é o elemento <input>. Ele pode ser exibido de diversas maneiras, dependendo do type atributo.

Código:

<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">

31.3 - O elemento

O elemento <label> define um rótulo para vários elementos do formulário. Ele é útil para usuários de leitores de tela, porque o leitor de tela lerá o rótulo em voz alta quando o usuário focar no elemento de entrada.

Também ajuda usuários que têm dificuldade em clicar em regiões muito pequenas (como botões de opção ou caixas de seleção) - porque quando o usuário clica no texto dentro do elemento <label>, ele alterna o botão de opção/caixa de seleção. O atributo for da tag <label> deve ser igual ao do elemento <input> para uni-los.

31.4 - O elemento <select>

O elemento <select> define uma lista suspensa:

Código:

<label for="cars">Escolha um carro:</label>
<select id="cars" name="cars">
   <option value="volvo">Volvo</option>
   <option value="saab">Saab</option>
   <option value="fiat">Fiat</option>
   <option value="audi">Audi</option>
</select>

Exemplo:


O elemento <option> define uma opção que pode ser selecionada. Por padrão, o primeiro item da lista suspensa é selecionado. Para definir uma opção pré-selecionada, adicione o atributo selected à opção:

<option value="fiat" selected>Fiat

Use o sizeatributo para especificar o número de valores visíveis:

Código:

<label for="cars">Esconlha um carro:
<select id="cars" name="cars" size="3">
   <option value="volvo">Volvo</option>
   <option value="saab">Saab</option>
   <option value="fiat">Fiat</option>
   <option value="audi">Audi</option>
</select>

Exemplo:


Use o multipleatributo para permitir que o usuário selecione mais de um valor:

Código:

<label for="cars">Esconlha um carro:</label>
<select id="cars" name="cars" size="4" multiple>
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="fiat">Fiat</option>
    <option value="audi">Audi</option>
</select>

31.5 - O elemento <textarea>

O elemento <textarea> define um campo de entrada multilinha (uma área de texto).

O atributo rows especifica o número visível de linhas em uma área de texto. O atributo cols especifica a largura visível de uma área de texto.

Código:

<textarea name="message" rows="10" cols="30">
The cat was playing in the garden.
</textarea>

Exemplo:


Você também pode definir o tamanho da área de texto usando CSS

Código:

<textarea name="message" style="width:200px; height:600px;">
The cat was playing in the garden.
</textarea>

31.6 - O elemento <button>

O elemento <button> define um botão clicável. Sempre especifique o typeatributo para o elemento botão. Diferentes navegadores podem usar diferentes tipos padrão para o elemento de botão.

Código:

<button type="button" onclick="alert('Hello World!')">Click Me!

31.7 - Os elementos <fieldset> e <legend>

O elemento <fieldset> é usado para agrupar dados relacionados em um formulário. O elemento <legend> define uma legenda para o elemento <fieldset>.

Código:

<form action="exemplos/actions.html">
  <fieldset>
    <legend>Personalia:</legend>
    <label for="fname">First name:</label><br>
    <input type="text" id="fname" name="fname" value="John"><br>
    <label for="lname">Last name:</label><br>
    <input type="text" id="lname" name="lname" value="Doe"><br><br>
    <input type="submit" value="Submit">
  </fieldset>
</form>

31.8 - O elemento <datalist>

O elemento <datalist> especifica uma lista de opções predefinidas para um elemento <input>.

Os usuários verão uma lista suspensa de opções predefinidas à medida que inserem os dados.

O atributo list do elemento <input> deve referir-se ao do elemento <datalist>.

Código:

<form action="exemplos/actions.html">
  <input list="browsers">
  <datalist id="browsers">
    <option value="Edge">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
  </datalist> </form>
