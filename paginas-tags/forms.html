29 - HTML Forms

29.1 - O elemento

Um formulário HTML é usado para coletar a entrada do usuário. A entrada do usuário é geralmente enviada a um servidor para processamento. O elemento HTML <form> é usado para criar um formulário HTML para entrada do usuário. Ele é um contêiner para diferentes tipos de elementos de entrada, como: campos de texto, caixas de seleção, botões de opção, botões de envio, etc.

Estrutura:

<form>
.
form elements
.
</form>

Todos os diferentes elementos do formulário são abordados no capítulo HTML Form Elements

E todos os diferentes atributos do formulário são abordados no capítulo HTML Form Attributes
