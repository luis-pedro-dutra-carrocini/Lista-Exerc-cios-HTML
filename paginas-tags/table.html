13 - HTML Table

Tabelas HTML

As tabelas HTML permitem que os desenvolvedores da web organizem os dados em linhas e colunas.

Definir uma tabela HTML

Uma tabela em HTML consiste em células de tabela dentro de linhas e colunas. Uma célula de tabela pode conter todos os tipos de elementos HTML: texto, imagens, listas, links, outras tabelas, etc.

Para melhor entendimento exemplos veja as definições dos componentes da tabela:

td - Significa dados da tabela.
tr - Significa linha da tabela.
th - Significa cabeçalho da tabela.

Código Básico:

<table>
    <tr>
       <th>Company</th>
       <th>Contact</th>
       <th>Country</th>
    </tr>
    <tr>
       <td>Alfreds Futterkiste</td>
       <td>Maria Anders</td>
       <td>Germany</td>
    </tr>
    <tr>
       <td>Centro comercial Moctezuma</td>
       <td>Francisco Chang</td>
       <td>Mexico</td>
    </tr>
</table>

Você pode melhorar a sua aparência utilizando o CSS. Veja a explicação de alguns atributos:

border - Para adicionar uma borda nos elementos da tabela.
border-collapse: collapse - Para evitar bordas duplas. background-color - Definir uma cor de fundo para cada elemento da tabela.

Código CSS:

table{
text-align: center;
border: 1px solid black;
border-collapse: collapse;
}

td, th{
padding: 10px;
background-color: #96D4D4;
border: 1px solid black;
}
