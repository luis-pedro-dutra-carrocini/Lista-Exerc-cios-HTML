25 - HTML Emojis

25.1 - O que são emojis?

Emojis parecem imagens ou ícones, mas não são. Na verdade são letras (caracteres) do conjunto de caracteres UTF-8 (Unicode). O UTF-8 cobre quase todos os caracteres e símbolos do mundo.

Para exibir uma página HTML corretamente, um navegador da Web deve conhecer o conjunto de caracteres usado na página.

Isso é especificado na tag <meta>: <meta charset="UTF-8">

25.2 - Caracteres UTF-8

Muitos caracteres UTF-8 não podem ser digitados em um teclado, mas sempre podem ser exibidos usando números (chamados de números de entidade):

Código:

Exibiremos as letras: &#65; &#66; &#67;

Exemplo:

Exibiremos as letras: A B C

O elemento <meta charset="UTF-8"> define o conjunto de caracteres. Os caracteres A, B e C são exibidos pelos números 65, 66 e 67.

Para que o navegador entenda que você está exibindo um caractere, você deve iniciar o número da entidade com & e terminar com ; (ponto e vírgula).


25.3 - Personagens emojis

Emojis também são caracteres do alfabeto UTF-8:

Código:

<h1>Meus Emojis</h1>
<p style="font-size:48px">
&#128512; &#128516; &#128525; &#128151;
</p>
