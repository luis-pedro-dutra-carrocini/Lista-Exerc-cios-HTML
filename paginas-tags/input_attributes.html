33.1 - Atributo Value

O atributo value especifica um valor inicial para um campo de entrada:

Código:

<form>
   <label for="fname">Nome:</label><br>
   <input type="text" id="fname" name="fname" value="John"><br>
   <label for="lname">Sobrenome:</label><br>
   <input type="text" id="lname" name="lname" value="Doe">
</form>

33.2 - Atributo Readonly

O atributo readonly especifica que um campo de entrada é somente leitura. Um campo de entrada somente leitura não pode ser modificado (no entanto, um usuário pode tabulá-lo, destacá-lo e copiar o texto dele). O valor de um campo de entrada somente leitura será enviado ao enviar o formulário!

Código:

<form>
   <label for="fname">Nome:</label><br>
   <input type="text" id="fname" name="fname" value="John" readonly><br>
   <label for="lname">Sobrenome:</label><br>
   <input type="text" id="lname" name="lname" value="Doe">
</form>

33.3 - Atributo Disabled

O atributo input disabled especifica que um campo de entrada deve ser desabilitado. Um campo de entrada desabilitado é inutilizável e não clicável. O valor de um campo de entrada desabilitado não será enviado ao enviar o formulário!

Código:

<form>
   <label for="fname">Nome:</label><br>
   <input type="text" id="fname" name="fname" value="John" disabled><br>
   <label for="lname">Sobrenome:</label><br>
   <input type="text" id="lname" name="lname" value="Doe">
</form>

33.5 - Atributo maxlength

O atributo

maxlength

especifica o número máximo de caracteres permitidos em um campo de entrada.

Quando o maxlength é definido, o campo de entrada não aceitará mais do que o número especificado de caracteres. No entanto, este atributo não fornece nenhum feedback. Portanto, se quiser alertar o usuário, você deve escrever o código JavaScript.

Código:

<form>
   <label for="fname">Nome:</label><br>
   <input type="text" id="fname" name="fname" size="50"><br>
   <label for="pin">PIN:</label><br>
   <input type="text" id="pin" name="pin" maxlength="4" size="4">
</form>

33.6 - Atributos min e max

Os atributos min e max, especificam os valores mínimo e máximo para um campo de entrada. Eles funcionam com os seguintes tipos de entrada: número, intervalo, data, data e hora local, mês, hora e semana. Use os atributos max e min juntos para criar um intervalo de valores legais.

Código:

<form>
   <label for="datemax">Data antes de 1980:</label>
   <input type="date" id="datemax" name="datemax" max="1979-12-31"><br><br>

   <label for="datemin">Data depois de 2000:</label>
   <input type="date" id="datemin" name="datemin" min="2000-01-02"><br><br>

   <label for="quantity">Quantidade (de 1 a 5):</label>
   <input type="number" id="quantity" name="quantity" min="1" max="5">
</form>

33.16 - Tipo Hidden

Define <input type="hidden"> um campo de entrada oculto (não visível para o usuário).

Um campo oculto permite que os desenvolvedores web incluam dados que não podem ser vistos ou modificados pelos usuários quando um formulário é enviado. Geralmente armazena qual registro do banco de dados precisa ser atualizado quando o formulário é enviado.

Embora o valor não seja exibido ao usuário no conteúdo da página, ele fica visível (e pode ser editado) usando as ferramentas de desenvolvedor de qualquer navegador ou a funcionalidade "Visualizar código-fonte". Não use entradas ocultas como forma de segurança!

Código:

<form>
   <label for="fname">Nome:</label>
   <input type="text" id="fname" name="fname"><br><br>
   <input type="hidden" id="custId" name="custId" value="3487">
   <input type="submit" value="Submit">
</form>

33.19 - Restrições de entrada

Aqui está uma lista de algumas restrições de entrada comuns:

Atributo Descrição
checked Especifica que um campo de entrada deve ser pré-selecionado quando a página for carregada (para type="checkbox" ou type="radio")
disabled Especifica que um campo de entrada deve ser desabilitado
max Especifica o valor máximo para um campo de entrada
maxlength Especifica o número máximo de caracteres para um campo de entrada
min Especifica o valor mínimo para um campo de entrada
pattern Especifica uma expressão regular para verificar o valor de entrada
readonly Especifica que um campo de entrada é somente leitura (não pode ser alterado)
required Especifica que um campo de entrada é obrigatório (deve ser preenchido)
size Especifica a largura (em caracteres) de um campo de entrada
step Especifica os intervalos numéricos legais para um campo de entrada
value Especifica o valor padrão para um campo de entrada

33.20 - Tipo Range

<input type="range"> define um controle para inserir um número cujo valor exato não é importante (como um controle deslizante). O intervalo padrão é de 0 a 100. No entanto, você pode definir restrições sobre quais números são aceitos com os atributos min, max e step:

Código:

<form>
   <label for="vol">Volume (de 0 a 50):</label>
   <input type="range" id="vol" name="vol" min="0" max="50">
</form>
