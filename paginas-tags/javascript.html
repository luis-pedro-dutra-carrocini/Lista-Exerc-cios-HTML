20 - HTML JavaScript

20.1 - Sintaxe

JavaScript torna as páginas HTML mais dinâmicas e interativas.

A tag HTML <script> é usada para definir um script do lado do cliente (JavaScript).

Os usos comuns do JavaScript são manipulação de imagens, validação de formulários e alterações dinâmicas de conteúdo.

Para selecionar um elemento HTML, o JavaScript geralmente usa o document.getElementById()método.

Este exemplo de JavaScript altera algumas propriedades de um elemento HTML com id="demo", ao clicar em alguns bootões:

Código:

<script> function mens(){
document.getElementById("demo").innerHTML = "Hello JavaScript!";
}

function font(){
document.getElementById("demo").style.fontSize = "25px";
}

function color(){
document.getElementById("demo").style.color = "red";
}

function colorback(){
document.getElementById("demo").style.backgroundColor = "yellow";
}

function resetar(){
document.getElementById("demo").style.backgroundColor = "#DAD7D7";
document.getElementById("demo").style.color = "black";
document.getElementById("demo").style.fontSize = "15px";
document.getElementById("demo").innerHTML = "";
}
</script>

<label id="demo"></label><br><br>
<button onclick="mens()">Exibir mensagem</button>
<button onclick="font()">Mudar Tamanho</button>
<button onclick="color()">Mudar Cor</button>
<button onclick="colorback()">Mudar Fundo</button>
<button onclick="resetar()">Resetar</button>

20.2 - Tag HTML <noscript>

A tag HTML <noscript> define um conteúdo alternativo a ser exibido aos usuários que possuem scripts desabilitados em seus navegadores ou possuem um navegador que não suporta scripts:

Código:

<script>
document.getElementById("demo2").innerHTML = "Hello JavaScript!";
</script>
<noscript>Desculpe, seu navegador não suporta JavaScript!</noscript>
